== Description ==

"**All-in-One Messenger**" is the only completely **free WordPress plugin** that lets you:

* manage conversations with customers;
* acquire messages from any channel (social networks, contact forms, etc...);
* allow more users to manage conversations;
* have a professional **chatbot** that asks the questions for you;
* avoid providing any roles on your social network pages;

directly from the WordPress dashboard without including other scripts to your website.

The "**All-in-One Messenger**" plugin has been designed to allow the management of customer service easily to everyone, without any technical knowledge or programming languages. Respecting the true nature of CMS (speed and ease of use).

This plugin is perfect for: **individual companies**, **small companies**, **medium-sized companies**, **large companies** and anyone else!

== Installation ==

To **automatically install** the "All-in-One Messenger" plugin you can search from the "Plugins > Add new" section of your WordPress dashboard and click on the "Install Now" button corresponding to this plugin.

To **manually install** the "All-in-One Messenger" plugin you can download the latest version from the WordPress.org site, and unpack the zip file in the "wp-content/plugins" folder of your CMS.

Now on the "Plugins" section of your WordPress dashboard you can see the "All-in-One Messenger" plugin, click on "Activate".
